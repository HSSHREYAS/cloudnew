# Smart Cloud Cost Estimator & Optimization Advisor

Production-style monorepo implementation of a stateless AWS cost estimation platform with:

- FastAPI backend with strict service-specific request validation
- AWS Pricing API and EC2 Spot integration boundaries
- Deterministic estimation and separate rule-based recommendations
- React SPA with Firebase authentication, dynamic forms, charts, and comparison workflows

## Structure

- `backend/` FastAPI service
- `frontend/` React application
